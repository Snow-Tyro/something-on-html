Категории контента

Поговорим о реструктуризации HTML-документа. В старой версии, было два типа элементов: блочные и строчные. В HTML5 введены семь основных моделей контента.

  • Метаданные
  • Встроенный контент
  • Интерактивный контент
  • Заголовочный контент
  • Фразообразующий контент
  • Потоковый контент
  • Секционирующий контент

Модели контента

Метаданные: Контент, который определяет представление или поведение остального контента на веб-странице. Эти элементы находятся в элементе head документа.

Элементы: base, link, meta, noscript, script, style, title

Второстепенный контент: Контент, который импортирует другие ресурсы в документ.

Элементы: audio, video, canvas, iframe, img, math, object, svg

Интерактивный контент: Контент, специально преданзначенный для взаимодействия с пользователем.

Элементы: a, audio, video, button, details, embed, iframe, img, input, label, object, select, textarea

Заголовочный контент: К этой модели принадлежат несколько строчных элементов, по анологии с HTML4.

Элементы: img, span, strong, label, br /, small, sub и прочие.
