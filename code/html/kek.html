    Применим элемент span для слова важно. Данный элемент определяет строчную секцию элемента. Также, строчные элементы можно пихать в блочные, но блочные в строчные нет.

    Поговорим об элементе form. Формы в HTML используются для сбора информации от пользователя. Формы создаются с помощью элемента form. Он имеет атрибут action, который указывает на веб-сраницу, которая должна загрузиться после отправки формы пользователем.

    Атрибут method указывает на метод HTTP-запроса (GET или POST), который будет использоваться при отправке формы

    Когда используется метод GET, в URL-адресе веб-страницы будут отображаться данные формы. Метод POST используется, когда требуется обновить данные в форме содержаться конфиденциальная иформация (пароли). Метод POST безопасней, потому что передаваемые данные не видны пользователю в URL-адресе страницы.

    Для того, чтобы получить данные пользователя, нужны соответствующие элементы формы, такие как текстовые поля. Элемент input имеет множество вариаций, что определяется атрибутом type. Например: text, password, radio, URL, submit и другое.
